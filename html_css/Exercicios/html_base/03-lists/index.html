Listas ordenadas

Definida por < ol > e seus itens definidos por < li >, dentro do seu elemento podemos utilizar os atributos do tipo de ordenação, como 'a' para letras minusculas, 'A' para letras maiusculas 'i' para algarismos romanos minusculos 'I' para algarismos romanos maiusculos e '1' indica os numeros (padrão)
